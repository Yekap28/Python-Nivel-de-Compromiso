Este repositorio contiene los archivos de la actividad 5 de la materia Fundamentos de Programacion, del segundo semestre de Ingenieria de Sistemas
La actividad me indica:
Se necesita una herramienta para evaluar el nivel de compromiso de cada sesión. 
Requisitos de Desarrollo: - Datos Iniciales: Una matriz con al menos 5 filas de datos. - Módulos: Se requiere un módulo (función) para calcular la 
clasificación de compromiso de una sesión basándose en su duración y clics. 

Lógica de Negocio: 
✓ Clasificar como "Alto" (si Duración > 180s y Clics > 8). 
✓ Clasificar como "Bajo" (si Duración < 60s o Clics < 3). 
✓ Clasificar como "Medio" en todos los demás casos. - Salida: Generar un informe listando el ID del cliente y su 
clasificación final.
